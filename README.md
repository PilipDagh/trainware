# trainware

A six-player, top-down survival game set on an 1870s steam train. Mine ore, fight raiders, refuel the engine, and reach the final stop together.

## Run it

```bash
npm install
npm start
```

Open `http://localhost:3000`, pick a name, and create or join a lobby.
