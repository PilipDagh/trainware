---
title: Controls
description: Keyboard, mouse, and on-screen touch controls for movement, combat, and interactions.
---

Controls are split between desktop (keyboard and mouse) and mobile (on-screen buttons). Mobile controls appear automatically on touch devices, or you can force them on from the **Settings** menu before starting a game.

## Desktop controls

| Action | Input |
| --- | --- |
| Move | `W`, `A`, `S`, `D` |
| Aim | Mouse cursor |
| Shoot | Left mouse button |
| Reload | `R` |
| Interact (engine, shop, coal car, barrels, horses) | `E` |
| Throw bomb (toward cursor) | `F` |
| Knife stab | `Spacebar` |
| Place beer barrel | `B` |
| Cycle spectator target (after death) | Spectator menu button |

## Mobile controls

Touch devices show a virtual joystick on the left and action buttons on the right. The on-screen buttons mirror the desktop bindings:

- **Shoot**, **Reload**, **Interact**, **Bomb**, and **Knife** buttons replace mouse and key inputs.
- Pinch-to-zoom and accidental scrolling are disabled in-game so touch input stays on the controls.

<Tip>
  To play on a desktop with a touch screen using on-screen controls, open the **Settings** menu and enable **Force mobile controls** before starting a game.
</Tip>

## HUD reference

The in-game HUD shows:

- **Magazine ammo** and **reserve ammo** as separate counters, so you always know whether to reload or restock at the shop.
- A live tally of **gold**, **silver**, and **coal** in your inventory.
- The current **fuel level** of the train and the active town/wilderness state.

The canvas and HUD stay hidden behind the menu until the game actually begins, so you no longer see an empty world while picking a lobby.
