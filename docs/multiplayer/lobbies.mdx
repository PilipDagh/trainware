---
title: Lobbies
description: Create public or private lobbies for up to six players, set passwords, and manage your roster before launching a run.
---

Lobbies are the staging area for every multiplayer run. You set a name, a player cap, and an optional password before starting the game.

## Create a lobby

<Steps>
  <Step title="Set your player name">
    On the main menu, type a name (up to 12 characters) into the **Name** field. Each player in a lobby is assigned a unique color shown in the roster.
  </Step>
  <Step title="Open the create form">
    Select **Create Lobby**.
  </Step>
  <Step title="Configure the lobby">
    | Field | Description | Default |
    | --- | --- | --- |
    | **Lobby Name** | Display name shown in the public list. | `My Train` |
    | **Max Players** | Cap from 1 to 6 players. | `6` |
    | **Password** | Leave blank for a public lobby, or set a password to make it private. | _empty_ |
  </Step>
  <Step title="Create and wait">
    Select **Create**. You're moved into the lobby room, where the player list updates as others join. When everyone is ready, select **Start Game**.
  </Step>
</Steps>

## Join a lobby

1. From the main menu, select **Join Lobby**.
2. The lobby list refreshes automatically. Each entry shows the lobby name, current player count, max player count, and a lock icon for private lobbies.
3. Select **Join**. If the lobby is private, enter the password when prompted.

<Note>
Private lobbies still appear in the public list with a lock icon, but you need the password to enter. To hide a lobby entirely, share the password out of band and treat it as the access control.
</Note>

## Public vs. private lobbies

| Behavior | Public | Private |
| --- | --- | --- |
| Visible in lobby list | Yes | Yes (with lock icon) |
| Requires password | No | Yes |
| Best for | Open drop-in games | Friends-only runs |

## Player roster

Once you're in a lobby, the roster lists every connected player by name in their assigned color. The host (the player who created the lobby) sees the same view. Anyone in the lobby can select **Start Game** to launch the run for the whole group.
