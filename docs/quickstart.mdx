Get a local server running and join your first run.

## Prerequisites

- Node.js 18 or later
- A modern browser (Chrome, Firefox, Safari, or Edge)

## Run the server

<Steps>
  <Step title="Install dependencies">
    From the repository root, install Express and Socket.IO:

    ```bash
    npm install
    ```
  </Step>
  <Step title="Start the server">
    ```bash
    npm start
    ```

    The server listens on port `3000` by default.
  </Step>
  <Step title="Open the game">
    Visit [http://localhost:3000](http://localhost:3000) in your browser. Share the URL (or your machine's IP) with friends on the same network to play together.
  </Step>
</Steps>

## Start your first run

1. Enter a name on the main menu.
2. Select **Create Lobby**, set a player limit and optional password, then select **Create**.
3. Wait for friends to join, then select **Start Game**.

<Tip>
Leave the password blank to make your lobby public. Public lobbies appear in the **Join Lobby** list for any player on the server.
</Tip>
